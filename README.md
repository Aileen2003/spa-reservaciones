# Spa Reservaciones (Orchid Spa)

Aplicación web para la gestión de reservaciones de un spa.  
Permite registrar, visualizar y cancelar citas utilizando un backend en Node.js con Express y una base de datos MySQL.  
El proyecto está contenedorizado con Docker y Docker Compose para facilitar su despliegue.

---

## Tecnologías utilizadas

- Node.js
- Express
- MySQL
- Docker
- Docker Compose
- HTML
- CSS

---

## Requisitos previos

Antes de ejecutar el proyecto es necesario contar con:

- Node.js (v18 o superior)
- Docker Desktop
- Docker Compose
